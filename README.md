# freym

MVP: a live feed of AI creators on Threads — their images and the prompts behind them.

- `supabase/functions/scraper-run` — scrapes Threads creators via ScrapeCreators, extracts prompts with Claude Haiku (incl. creator replies / snippet attachments)
- `supabase/functions/scraper-feed` — public JSON feed for the site
- `docs/index.html` — static site (GitHub Pages)
